# Lab 07 Guide: Network Protocols

This folder contains a self-contained version of the lab on network protocols, routing, and reliable data transmission.

## Contents
- [explanation.md](explanation.md) with the theoretical summary.
- [guide.md](guide.md) with the recommended path for guiding students.
- [review-sheet.md](review-sheet.md) with a ready-to-use handout and review questions.
- [solutions/](solutions) with the question PDF and answer sheet.

## Lab goal
Understand how the network layers work together: addressing, switching, routing, reliability, and application services such as HTTP and TLS.

## Quick start
There are no files to execute: this lab is theoretical. The folder serves as a complete package for explanation, review, and classroom questions.
